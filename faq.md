## FAQ — Često Postavljana Pitanja

### Šta je LugZe?

LugZe (Linux User Group of Zenica) je zajednica zeničkih Linuxaša, osnovana 2002. godine. Više o nama na stranici [O nama]({{ "/o-nama/" | relative_url }}).

### Gdje mogu pratiti aktivnosti grupe?

Naš GitHub repo dokumentuje posljednje aktivnosti i izmjene na tekstovima:

[LugZe GitHub commits](https://github.com/lugze/blog/commits/master)

### Kako se mogu pridružiti?

Pridruži se WhatsApp grupi **„Linux kafa"** — tu se okupljaju stari i novi članovi, razmjenjuje znanje i dogovaraju druženja uz kafu. Javi se bilo kojem članu ili nas kontaktiraj putem GitHub-a.

### Kako mogu doprinijeti blogu?

Blog je open-source i hostovan na GitHub-u. Možeš:

1. Forkati [repozitorij](https://github.com/lugze/blog)
2. Dodati novi post u `_posts/` folder (format: `YYYY-MM-DD-naslov.markdown`)
3. Poslati pull request

Postovi se pišu u Markdownu. Pogledaj postojeće postove za primjer formata.

### Koji su tehnički zahtjevi za pokretanje bloga lokalno?

```
gem install bundler jekyll
git clone https://github.com/lugze/blog.git
cd blog
bundle install
bundle exec jekyll serve
```

Blog će biti dostupan na `http://localhost:4000/blog/`.

### Moram li znati programirati da bih se pridružio?

Ne. LugZe okuplja ljude raznih nivoa znanja — od početnika koji tek instaliraju Linux do iskusnih administratora. Jedini preduslov je znatiželja i želja za učenjem. I kafica, naravno. ☕
